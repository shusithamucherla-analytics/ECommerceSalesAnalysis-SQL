# UK E-Commerce Sales Analysis using SQL Server

 Overview
This project explores a real-world UK based online retail dataset 
containing 541,909 transactions recorded between 2010 and 2011. 
The analysis was performed using SQL Server to identify sales trends, 
top performing products, customer behavior, and monthly revenue patterns.

 Tools and Technologies
- SQL Server
- SQL Server Management Studio (SSMS)
- Dataset: UCI Online Retail Dataset (Kaggle)

 What I Analyzed
- Order distribution across different countries
- Top 10 best selling products by quantity sold
- Monthly revenue trends across the entire dataset
- Top 10 customers ranked by total revenue generated
- Overall average, maximum, and minimum order values

 Key Findings
- The United Kingdom dominates order volume, representing 
  the majority of all transactions
- World War 2 Gliders Asstd Designs emerged as the 
  top selling product by quantity
- November 2011 recorded peak revenue, consistent with 
  holiday season shopping behavior
- A single customer (ID: 14646) generated the highest 
  cumulative revenue
- The average order value was £18.06, with the highest 
  single order reaching £77,183.60

 Dataset Source
UCI Online Retail Dataset — available on Kaggle
